размышления о структуре факторов


.1. существует познавательно-творческий потенциал

    .1.1. часть познания - наблюдения и диалектика


    .1.2. часть творчества:
        .компоновка и преобразование информации.
        .постановка задач, формирование моделей, планирование.


    .1.3. тут разсматриваю только информационные проявления, свойственные цепочке:
        .различение => внимание => интеллект:
        	.«первичная» информация, даваемая в Различении,
        		+ информация из памяти
        		+ «оперативная» информация
        		+ «ответная» информация, обусловленная прошлыми действиями
        	=>
        	.внимание и осознанно волевая команда на осмысление информации
        	=>
        	.интеллект
        	=>
       		.поведение во внутреннем мире (переосмысление прошлого, намерения на будущее,
       		    формирование мировоззрения и миропонимания)
        	=>
        	.поведение во внешнем мире (воздействие на среду)
        	=>
        	.«ответная» информация.

        .при этом в концепцию (отображаемую concept_viewer-ом) выводиться лишь часть информации,
        необходимая для поддержания процессов, на которые цепочка указывает.


    .1.4. из сказанного выше разворачиваются:
        .набор видов записей.
        .последовательность/процедура заполнения записей разных видов - заполениня "пустых полочек".
        .единицы учёта и адресации информации.
        .набор вариантов указания адресов информации и отображения адресуемой информации (цитирования).
        .требование неизменяемости фрагментов и способ версионирования.
